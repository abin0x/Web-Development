Learning Basic CSS

CSS is a powerful tool for web developers, allowing them to create visually appealing and well-structured web pages. Understanding how to use CSS effectively is essential for modern web design.

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of web pages.

In this example, we will explore how to apply colors to HTML elements using CSS.

Applying Colors

Colors can be applied to HTML elements in various ways:

  • Using color names (e.g., "red", "blue")
  • Using hexadecimal values (e.g., "#ff0000" for red)
  • Using RGB values (e.g., "rgb(255, 0, 0)" for red)
  • Using RGBA values (e.g., "rgba(255, 0, 0, 0.5)" for semi-transparent red)
  • Using HSL values (e.g., "hsl(0, 100%, 50%)" for red)
